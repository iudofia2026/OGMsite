## Color Palette

### Primary Brand Colors

The OGM brand is built on a sophisticated four-color palette that balances bold sophistication with approachable warmth.

#### Black
- **Hex:** `#000000`
- **RGB:** rgb(0, 0, 0)
- **Usage:** Primary text, luxury accents, premium backgrounds
- **Psychology:** Sophistication, elegance, exclusivity

#### Red
- **Hex:** `#CC071E`
- **RGB:** rgb(204, 7, 30)
- **Usage:** Call-to-action buttons, highlights, energy accents
- **Psychology:** Passion, energy, boldness

#### Teal
- **Hex:** `#02BBB7`
- **RGB:** rgb(2, 187, 183)
- **Usage:** Secondary elements, fresh accents, modern touches
- **Psychology:** Freshness, balance, contemporary appeal

#### Beige/Tan
- **Hex:** `#C9A55B`
- **RGB:** rgb(201, 165, 91)
- **Usage:** Premium accents, warmth backgrounds, sophisticated highlights
- **Psychology:** Warmth, approachability, craft heritage

### Color Applications

**Primary Combinations:**
- Black + Beige: Premium luxury aesthetic
- Teal + White: Fresh, modern appeal
- Red + Black: Bold, energetic statements
- Beige + Teal: Warm, contemporary balance

**Usage Guidelines:**
- Use Black as primary text color on light backgrounds
- Use Beige for premium accents and sophisticated highlights
- Use Red sparingly for CTAs and important actions
- Use Teal for fresh, modern secondary elements
- Maintain 70% Black/White, 20% Beige, 10% Accent colors ratio

## Typography

### Primary Font Family: Goldenbook

**Font Type:** Serif
**Role:** Headlines, titles, premium text
**Available Weights:**
- Light (300)
- Regular (400)
- ExtraBold (700)
- Heavy (800)
- Black (900)

**Usage:**
- Hero headlines and titles
- Product names
- Premium editorial content
- Brand statements

**Characteristics:**
- Elegant, sophisticated serif design
- Excellent readability at large sizes
- Premium luxury aesthetic
- Strong editorial presence

### Secondary Font Family: Raleway

**Font Type:** Sans-serif
**Role:** Body text, UI elements, secondary content
**Available Weights:**
- Thin (100)
- ExtraLight (200)
- Light (300)
- Regular (400)
- Medium (500)
- SemiBold (600)
- Bold (700)
- ExtraBold (800)
- Heavy (900)

**Usage:**
- Body copy and paragraphs
- Navigation and UI elements
- Subtitles and descriptions
- Button and link text

**Characteristics:**
- Clean, modern geometric design
- Excellent readability across sizes
- Versatile weight range
- Contemporary sophistication

### Typography Scale

#### Desktop
```css
--font-size-hero: clamp(4rem, 12vw, 12rem);
--font-size-h1: clamp(2.5rem, 6vw, 4rem);
--font-size-h2: clamp(1.8rem, 4vw, 2.5rem);
--font-size-h3: clamp(1.5rem, 3vw, 2rem);
--font-size-subtitle: clamp(1.2rem, 3vw, 2rem);
--font-size-body: clamp(1rem, 2.5vw, 1.25rem);
--font-size-caption: clamp(0.875rem, 2vw, 1rem);
--font-size-small: clamp(0.75rem, 1.5vw, 0.875rem);
```

#### Mobile
```css
--font-size-hero: clamp(3rem, 15vw, 8rem);
--font-size-h1: clamp(2rem, 8vw, 3rem);
--font-size-h2: clamp(1.5rem, 6vw, 2rem);
--font-size-subtitle: clamp(1rem, 4vw, 1.5rem);
--font-size-body: clamp(0.95rem, 3vw, 1.15rem);
```

### Typography Rules

1. **Hierarchy:** Use Goldenbook for headlines, Raleway for body text
2. **Contrast:** Maintain proper contrast ratios (WCAG AA minimum)
3. **Spacing:** Use generous line height for luxury feel (1.4-1.6)
4. **Weight:** Use font weights deliberately - avoid too many variations
5. **Alignment:** Left-align for readability, center for impact

## Visual Elements

### Logo Usage

**Primary Logo:** OGM square logo (black)
**Secondary Logo:** OGM with text (horizontal)

**Usage Guidelines:**
- Minimum clear space: 1x logo height
- Minimum size: 120px width (digital), 1 inch (print)
- Background colors: Black, white, or beige
- Never stretch or distort logo proportions

### Imagery Style

**Product Photography:**
- Clean, minimalist backgrounds
- 75% opacity for subtle, sophisticated look
- Black and white aesthetic with color accents
- Premium lighting and composition

**Background Images:**
- Grayscale filter for unified aesthetic
- Subtle opacity for layered depth
- High contrast for text readability
- Premium, sophisticated subjects

### Design Principles

1. **Minimalism:** Clean, uncluttered layouts
2. **Sophistication:** Premium materials and finishes
3. **Versatility:** Works across digital and print
4. **Consistency:** Unified visual language
5. **Quality:** High-resolution, professional execution

## Brand Consistency

### Do's

✅ Use the official color palette exactly
✅ Maintain proper typography hierarchy
✅ Follow spacing and layout guidelines
✅ Use high-quality images and graphics
✅ Keep designs clean and uncluttered
✅ Test across all platforms and devices

### Don'ts

❌ Modify colors or use unapproved shades
❌ Stretch or distort logos and text
❌ Use unauthorized fonts or weights
❌ Crowd designs with too many elements
❌ Ignore mobile responsiveness
❌ Compromise on image quality
