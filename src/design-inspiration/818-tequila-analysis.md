# 818 Tequila Website - Premium Branding Analysis

Based on analysis of the 818 Tequila website (Kendall Jenner's premium tequila brand), this document outlines the key design elements that contribute to their super premium positioning.

## üé® Typography Analysis

### Primary Font Stack

#### 1. **GravityVariable (Custom Brand Font)**
- **Usage**: Primary headings, hero titles, brand name
- **Weight**: 900 (Black/Heavy)
- **Characteristics**:
  - Modern, clean sans-serif
  - Variable font technology for precise weight control
  - Strong presence, confident personality
  - Excellent for large-scale typography

#### 2. **Sackers Gothic Std (Premium Serif)**
- **Weights Available**: Light (300), Medium (500), Heavy (900)
- **Usage**: Secondary headings, premium text elements
- **Characteristics**:
  - Classic, sophisticated serif
  - Art Deco influence
  - Premium, luxury feel
  - High-end editorial style

#### 3. **Acumin Extra Condensed**
- **Weight**: 400 (Regular)
- **Usage**: Body text, navigation, supporting content
- **Characteristics**:
  - Clean, modern sans-serif
  - Excellent readability
  - Space-efficient condensed design
  - Professional, contemporary

#### 4. **CaslonPro (Classic Serif)**
- **Weight**: 400 (Regular)
- **Usage**: Long-form content, descriptions
- **Characteristics**:
  - Traditional, readable serif
  - Classic book typography feel
  - Elegant, timeless
  - High readability for body text

#### 5. **ABCFavoritMono (Monospace)**
- **Weight**: 500 (Medium)
- **Usage**: Technical information, labels, details
- **Characteristics**:
  - Modern monospace font
  - Clean, technical aesthetic
  - Good for data/specifications

### Typography Hierarchy Strategy

1. **Hero Level**: GravityVariable (900) - Bold brand statements
2. **Heading Level**: Sackers Gothic Std - Premium section titles
3. **Body Level**: Acumin Extra Condensed - Clean, readable content
4. **Editorial Level**: CaslonPro - Storytelling, descriptions
5. **Technical Level**: ABCFavoritMono - Details, specs
